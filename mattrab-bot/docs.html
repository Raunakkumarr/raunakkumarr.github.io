Instructions to Use.

Public Commands

Anyone on the server can use these commands to get a reponse from the bot.

  1. &changeActivity :

    This will change the activity of bot to a random activity from the activity list.

  2. &sad :

    If someone sends this command, bot will return an inspirational quote or message back to the respective channel.

  3. &help :

    This wil send an embeded link to the bot's docs file or user manual.

  4. &clear amount :

    When someone sends '&clear amount', it will delete the latest amount number of messages, like if I send &clear 5, it will delete latest 5 messages from that particular channel.

  5. &rules :

    When someone sends '&rules' in any channel, Bot will return an embed of the list of rules to the specified rules channel.

  6. &chdivisions :

    When someone sends '&chdivisions' in any channel, Bot will return an embed of the list of channel division to the specified rules channel.

Note that all these commands except the changeActivity are case insensetive and starts with the '&' sign.

Inbuilt Commands

These are the inbuilt functions of the bot and cannot be altered by anyone.

  1. When someone joins the server, bot will automatically send random welcome message and a particularly designed welcome card or template to the welcome channel and tag the person's id.

  2. When someone sends any sad or depressing words like sad, depressed, etc. bot will automatically send inspirational message to the respective channel if it is provided permission to do so.

  3. When someone sends greetings to everyone in channel saying hello all or hi everyone, the bot will send him/her a response if it's provided the permission to text in that channel.

  4. When someone add reaction on a specific role clarification messsage with specified emojis, Bot will automatically assign them a specific role assigned to that emoji.

  5. When someone removes reaction on a specific role clarification messsage with specified emojis, Bot will automatically remove a specific role from them assigned to that emoji.
